# To-do-List
📝  Todo List with Local Storage
A simple and efficient Todo List application built with JavaScript, HTML, and CSS. This app allows users to add, delete, and store tasks using local storage, ensuring that todos persist even after the page is refreshed.

✨ Features
✅ Add new tasks effortlessly
🗑️ Delete tasks when completed
🔄 Persistent storage using local storage
🎨 Clean and user-friendly UI
